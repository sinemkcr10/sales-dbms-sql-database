# sales Database Project

This project is an sales database designed using SQL.

## Tables
- Customer
- Seller
- Product
- Category
- Address
- Order
- Order_Item
- Order_Status

## Features
- Relational database design
- Primary & Foreign keys
- Sample INSERT data
- UPDATE operations
- Real-life e-commerce scenarios

## How to Run
1. Create a database (PostgreSQL recommended)
2. Open `ecommerce.sql`
3. Execute all queries
